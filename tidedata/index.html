Use In Your App

JavaScript Example

// Get wave height from M2 buoy
async function getWaveHeight() {
    const url = 'https://erddap.marine.ie/erddap/tabledap/IWBNetwork.json' +
                '?station_id,SignificantWaveHeight,time' +
                '&station_id="M2"&time>=now-1hour';
    
    const response = await fetch(url);
    const data = await response.json();
    
    // Extract the wave height
    const rows = data.table.rows;
    if (rows.length > 0) {
        const latestReading = rows[rows.length - 1];
        const waveHeight = latestReading[1]; // SignificantWaveHeight column
        console.log(`Current wave height: ${waveHeight}m`);
    }
}

getWaveHeight();

Python Example

import requests
import pandas as pd

# Get tide data from Galway Harbor
url = "https://erddap.marine.ie/erddap/tabledap/IrishNationalTideGaugeNetwork.csv"
params = {
    "station_id": '"Galway Port"',
    "time>=": "now-1day"
}

# Fetch the data
response = requests.get(url, params=params)

# Load into pandas DataFrame
from io import StringIO
df = pd.read_csv(StringIO(response.text), skiprows=[1])

# Display latest tide level
latest_tide = df.iloc[-1]['Water_Level_LAT']
print(f"Current tide level: {latest_tide}m")

💡 Pro Tips:

  • • Use .json format for JavaScript/APIs
  • • Use .csv format for Excel/spreadsheets
  • • Use time>=now-7days for last week's data
  • • All times are in UTC
  • • Wave heights in IWBNetwork are in meters
